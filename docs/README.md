# EnobFinance — Documentação

Gerenciador de finanças pessoais construído em **Python + Django (full stack)**, organizado por mês, com template replicável, controle de cartões, investimentos com meta e gestão de pagamentos/vendas parcelados.

Parte do ecossistema **Enob** · Design System Enobtech.
